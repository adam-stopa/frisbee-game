# Frisbee Game

## Overview
A simple Python pygame-based game where you control "Laura" (a dog) that jumps to catch frisbees while avoiding tree obstacles.

## Project Structure
- `game.py` - Main game file with pygame implementation
- `assets/` - Game graphics (sprites for Laura, frisbee, trees)
- `test_game.py` - Test file
- `requirements.txt` - Python dependencies

## Running the Game
The game runs via the "Frisbee Game" workflow which displays in VNC mode.

### Controls
- **SPACE** - Jump
- **R** - Restart after game over

## Dependencies
- pygame 2.6.1
- pytest 7.4.3 (for testing)

## Development
This is a desktop GUI application using pygame, displayed through VNC in the Replit environment.
