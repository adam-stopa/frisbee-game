# Frisbee Game

Prosty projekt gry "Frisbee" w Pythonie.

## Opis
Krótki projekt demonstracyjny — prosty silnik gry i mechanika rzucania frisbee.

## Wymagania
- Python 3.8+

## Instalacja i uruchomienie
1. Stwórz i aktywuj wirtualne środowisko:

```bash
python3 -m venv venv
source venv/bin/activate
```

2. (Opcjonalnie) Zainstaluj zależności, jeżeli istnieje `requirements.txt`:

```bash
pip install -r requirements.txt
```

3. Uruchom grę:

```bash
python game.py
```
